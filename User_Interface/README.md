# UI Setup
## Install Dependencies
```bash
npm install
```

## Run Development Server
```bash
npm run dev
```

Frontend will run on `http://localhost:5173`

Make sure the backend is running on `http://localhost:8080` 

## Test the Application
1. Sign up with a `@umass.edu` email
2. Login with your credentials
3. Fill out the survey in the Profile page
4. View recommended roommates on the Dashboard (this will work if there are multiple users who have registered)
5. Browse all users in the Browse page
